# jTag 
Makes dynamic markup a breeze.

Introducing jTag, a dynamic way to create html, in javascript, that takes the pain out of the &lt; and &gt;

jTag is a library of the following functions that generate html markup:

* core
* jLabel
* jText
* jHidden
* jArea
* jSelect
* jCheck
* jRadio
* jSubmit
* jReset

* jDiv
* jTable
